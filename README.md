# Attendance System

## Project Description
This is a simple cloud-based Attendance System project created using HTML, CSS, and JavaScript.

The system allows users to:
- Enter student name
- Mark attendance
- View attendance records with date and time

## Technologies Used
- HTML
- CSS
- JavaScript

## Features
- Simple user interface
- Attendance tracking
- Date and time display
- Responsive design

## How to Run the Project
1. Download the project folder
2. Open the folder
3. Run the `index.html` file in browser
